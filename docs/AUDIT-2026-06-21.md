# Codebase Audit — 2026-06-21

> Prioritized list of remaining bugs, error-handling gaps, and hardening opportunities for `ai_video_editor`.

## Legend

- **Critical** — crashes, data loss, security issues, or broken end-to-end flows.
- **High** — runtime failures, incorrect behavior, or frequent user-facing errors.
- **Medium** — UX/robustness issues that degrade under load or edge cases.
- **Low** — cleanup, tech debt, and defensive improvements.
