# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these steps:

1. **Do NOT open a public issue.** Security vulnerabilities should be reported privately.
2. Email the maintainers at **security@ai-video-editor.dev** with:
   - A description of the vulnerability
   - Steps to reproduce (if applicable)
   - Potential impact assessment
   - Any suggested fixes or mitigations
3. Allow up to **72 hours** for an initial response.
4. We will work with you to validate the issue, develop a fix, and coordinate disclosure.

## Security Measures

- Dependencies are scanned weekly via GitHub Dependabot
- CodeQL analysis runs on every push to `main`
- `npm audit` is integrated into CI
- All PRs require review before merge

## Acknowledgments

We appreciate responsible disclosure and will acknowledge contributors in our changelog.
